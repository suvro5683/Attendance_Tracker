AnalyticsPerformance Tracker

Built entirely from the uploaded Reconciliation Report for each period — no other data source is used. Columns are read as: User → App Name, Points → Total Points, Initial Review → Total IR, Reconcile → Total Reco, Final Review → FR. The one derived figure is Average — for Analysts that's Total Points ÷ actual working days, for Specialists it's Total FR ÷ actual working days — with Leave Tracker dates automatically excluded either way.
